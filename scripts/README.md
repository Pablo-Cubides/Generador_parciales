# 🛠 Directorio de Scripts Utilitarios

Scripts en Python diseñados para ser invocados por el Agente de IA para realizar operaciones que requieren lógica determinística o acceso a servicios externos.

## Scripts

- **`supabase_client.py`**: Cliente centralizado para interactuar con la base de datos de Supabase.
- **`compile_exam.py`**: Lógica de permutación de opciones y generación de documentos Word (.docx) usando la librería `python-docx`.
- **`generate_analytics.py`**: Generación de reportes de errores y analítica pedagógica en formato tabular.
- **`schema.sql`**: Esquema de base de datos listo para ser inyectado en Supabase.

## Requisitos
Asegúrate de tener instaladas las dependencias:
`pip install supabase python-dotenv python-docx`
