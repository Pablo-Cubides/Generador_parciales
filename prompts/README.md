# 🧠 Directorio de Prompts Maestros

Este directorio contiene las instrucciones de alto nivel que definen el comportamiento de los agentes de IA dentro de este flujo de trabajo.

## Contenido

- **`generator_agent.md`**: Define cómo el agente debe extraer información académica de las fuentes (RAG) y transformarla en preguntas de opción múltiple (A-E) con rigor técnico.
- **`grader_agent.md`**: Instrucciones específicas para el procesamiento de imágenes y visión artificial, permitiendo al agente "leer" hojas de respuestas y asignar notas.

## Uso

Para usar estos prompts, simplemente indica al agente:
*"Usa las instrucciones de [nombre_del_prompt].md para realizar [acción]"*.
